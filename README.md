# crypto-msg-system
Elliptic-Curve-Cryptography-Based Secure Messaging System with Certificate Authority

## Implemented:
ECDSA with SHA-256  
ECIES with AES for symmetric key encryption
Tested with secp256k1

## Dependencies
"aes-js": "^3.1.0"
"big-integer": "^1.6.25"
"pbkdf2": "^3.0.14"
"sha.js": "^2.4.9"
